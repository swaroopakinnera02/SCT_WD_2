# Stopwatch Web Application

## Description

A simple and interactive stopwatch web application that allows users to measure time accurately. The application includes Start, Pause, Reset, and Lap functionalities with a clean and responsive user interface.

## Features

* Start, Pause, and Reset timer
* Record lap times
* Display time in Hours:Minutes:Seconds:Milliseconds format
* Responsive and user-friendly design

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the project.
2. Open `index.html` in a web browser.
3. Use the stopwatch controls to track time and record laps.
